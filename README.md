# Fin Tracking

A full-stack personal finance tracking web application that helps users manage their income, expenses, budgets, and financial overview in one place.

## 🚀 Features

- Add and manage income/expense transactions
- Set category-wise budgets
- Track spending against budget limits
- Dashboard with financial summaries
- Responsive and user-friendly UI
- Full-stack architecture with separate client and server

## 🛠 Tech Stack

### Frontend
- HTML / CSS / JavaScript
- Bootstrap / Custom CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB

⚙️ Installation
1. Clone Repository
git clone https://github.com/DRPatel75/fin-tracking.git
cd fin-tracking
2. Install Dependencies
npm install

Install client/server dependencies if separate:

cd client
npm install

cd ../server
npm install
▶️ Run Project
Start Backend
cd server
npm start
Start Frontend
cd client
npm start
📌 Environment Variables

Create a .env file inside server/ and add:

PORT=5000
DATABASE_URL=your_database_connection_string
JWT_SECRET=your_secret_key

(Remove JWT_SECRET if not using authentication)
